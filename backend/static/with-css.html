🎉 静态资源服务演示

这个页面使用了外部CSS文件，通过uvicorn静态文件服务提供

✅ 静态文件服务功能

uvicorn 不仅可以作为 API 服务器，还可以提供完整的 Web 服务功能：

  • HTML 页面服务
  • CSS 样式文件
  • JavaScript 脚本
  • 图片和媒体文件
  • 字体文件
  • 文档和 PDF 文件

🔧 技术特性

  • 高性能：基于 ASGI 标准，支持异步处理
  • 自动重载：开发模式下文件变化自动重启
  • CORS 支持：内置跨域资源共享支持
  • 静态文件：高效的静态资源服务
  • API 文档：自动生成 Swagger/OpenAPI 文档

📁 文件结构

backend/
├── static/          # 静态文件目录
│   ├── style.css    # CSS 样式文件
│   ├── demo.html    # HTML 页面
│   └── with-css.html
├── templates/       # 模板文件目录
└── app/
    └── main.py      # FastAPI 应用
